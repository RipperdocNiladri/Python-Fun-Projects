# 🎯 Number Guessing Game

A fun and interactive command-line Number Guessing Game built with **Python**. Test your luck, improve your guessing skills, earn coins, and challenge yourself with different difficulty levels!

---

## ✨ Features

* 🎮 Three difficulty levels

  * Easy (1–20)
  * Medium (1–50)
  * Hard (1–100)
* ❤️ Limited lives based on difficulty
* 💰 Coin reward system
* 🔥 Win streak tracker
* 🏆 High score tracking
* 📈 Helpful "Higher" and "Lower" hints
* 🎉 Random victory messages
* 🔄 Play again option
* 🛡️ Input validation for invalid entries

---

## 🛠️ Built With

* Python 3
* `random` module

---

## 🚀 How to Run

1. Clone this repository.

```bash
git clone https://github.com/RipperdocNiladri/python-fun-projects.git
```

2. Navigate to the project folder.

```bash
cd python-fun-projects
```

3. Run the game.

```bash
python number-guessing-game.py
```

---

## 🎮 Gameplay

1. Choose a difficulty level.
2. Guess the hidden number.
3. Use the **Higher** and **Lower** hints to get closer.
4. Win before you run out of lives.
5. Earn coins and build your winning streak.
6. Beat your high score!
