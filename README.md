## 📖 About

Welcome to my **Python Fun Projects** repository!

This repository contains small, practical, and enjoyable Python projects that I build while improving my programming skills. Each project focuses on learning core Python concepts through real coding rather than just theory.

Projects range from simple command-line applications to GUI programs, utilities, mini games, automation scripts, and much more.

As I continue learning Python, this repository will keep growing. 🚀
